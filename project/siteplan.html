Project Site Plan

Site Name & Purpose

Samuel Innocent | Data Science & Web Dev Portfolio

I chose this name because it is my personal professional brand. As a Data Scientist and Web Developer, my name is the "product." It clearly tells recruiters exactly who I am and what I do immediately upon landing on the page.

Purpose: This website will serve as a professional portfolio to showcase my technical skills (Data Science & Web Dev). It will feature an "About Me" section, a "Project Showcase" (Telco Churn & Energy Analysis), and a "Contact" area.

Scenarios

Questions my target audience (Recruiters & Employers) might ask:

  • "I see Samuel lists 'Data Science' on his resume; where can I see code examples of his Python Energy Analysis?"
  • "Does Samuel have any experience combining data backend logic with a frontend web interface?"
